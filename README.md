# Recording Desktop Client

Windows 환경에서 오디오 장치 제어와 녹음 흐름 관리를 지원하는 데스크톱 클라이언트 프로젝트입니다.

이 프로젝트는 단순한 UI 애플리케이션이 아니라,
**실시간 장치 상태, 녹음 제어, 서버 연동 흐름**을 함께 다루는 실무형 클라이언트 구조를 보여주는 데 목적이 있습니다.

## 프로젝트 개요

이 애플리케이션은 다음과 같은 역할을 수행합니다.

- 입력/출력 오디오 장치 탐색 및 제어
- 녹음 시작/종료 상태 관리
- 볼륨, 음소거, 오디오 레벨 모니터링
- 시스템 트레이 기반 동작
- 서버와 연결되는 녹음 워크플로우 처리

## 주요 포인트

- WPF 기반의 Windows 데스크톱 애플리케이션 구조
- NAudio를 활용한 오디오 장치 및 입력/출력 레벨 처리
- View / ViewModel / Service 분리를 의식한 구조
- 실제 사용 흐름을 고려한 트레이 앱 UX 및 상태 전환
- 로컬 장치 제어와 서버 연동을 함께 다루는 실무형 클라이언트 경험

## 기술 스택

- C#
- .NET / WPF
- NAudio
- Windows Audio Device API
- System Tray / Desktop UX

## 공개용 정리 기준

이 저장소는 포트폴리오 공개용으로 정리된 버전입니다.

다음과 같은 항목은 제외하거나 비공개 전제로 정리했습니다.

- 내부 설정 파일
- 로컬 DB 파일
- 실행 산출물(bin, obj)
- 로그 및 사용자별 로컬 파일
- 실제 운영 환경에 사용된 서버 정보 및 배포 설정

현재 공개본은 구조와 역할 설명을 중심으로 정리한 버전이며,
실제 운영 환경에 맞춘 상세 설정 및 내부 연동 정보는 포함하지 않습니다.

## 실행 관련 참고

프로젝트는 Windows 환경을 전제로 하는 WPF 애플리케이션입니다.
로컬 환경 및 의존성 상태에 따라 빌드/실행 설정이 추가로 필요할 수 있습니다.

## 포트폴리오 관점에서의 의미

이 프로젝트는 웹 시스템만이 아니라,
**실시간 장치 제어와 운영 흐름을 포함하는 데스크톱 애플리케이션을 다룬 경험**을 보여주는 사례로 활용하고 있습니다.
